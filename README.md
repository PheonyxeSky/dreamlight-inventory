\# Maleficent-Valley-Tracker



\*\*PhoenyxeSkye's Inventory: Dark, Deadly, and Dripping in Elegance\*\* 🔥✨



A sleek tracker for Disney Dreamlight Valley inventory, covering all DLCs (Eternity Isle, Storybook Vale, Wish Blossom, and more). Built for Maleficent queenies who demand beauty and power.



\## Features (Coming Soon)

\- Track currencies (dreamlight, moonstones, gold).

\- Inventory sections for main valley, DLCs, star paths.

\- CLI commands: add, subtract, list, save.

\- Maleficent mode: Highlights purple/dragon/thorn items.

\- Shareable with simple login for other players.



\## Setup

1\. Clone the repo.

2\. Install Python.

3\. Run `python track.py --help` for commands.
